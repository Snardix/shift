# File Content Filter Utility

## Описание

Консольная утилита на Java для фильтрации содержимого текстовых файлов.

Программа принимает несколько входных файлов, содержащих вперемешку:
- целые числа
- вещественные числа
- строки

Каждая строка считается отдельным элементом (разделитель — перевод строки).

Утилита распределяет данные по разным выходным файлам:
- integers.txt
- floats.txt
- strings.txt

Файлы создаются **только при наличии соответствующих данных**.

---

## Поддерживаемые опции

| Опция | Описание |
|-------|----------|
| `-o <path>` | Путь для выходных файлов |
| `-p <prefix>` | Префикс имён выходных файлов |
| `-a` | Режим добавления (append) |
| `-s` | Краткая статистика |
| `-f` | Полная статистика |

---

## Статистика

### Краткая (-s)
Выводит только количество элементов каждого типа.

### Полная (-f)

Для чисел:
- количество
- минимум
- максимум
- сумма
- среднее значение

Для строк:
- количество
- длина самой короткой строки
- длина самой длинной строки

---

##  Пример запуска
Компиляция javac Main.java
java Main -s -a -p sample- in1.txt in2.txt (примеры данных лежат в папке src)
